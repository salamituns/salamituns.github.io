Data Sources
  • HHS Office for Civil Rights Breach Portal - HIPAA breaches affecting 500+ individuals, 2015 to 2026
  • CMS Provider Data Catalog - Medicare-certified home health agencies, April 2026
Methodology

Breach data from the HHS OCR portal (archived + under-investigation reports) was cleaned, deduplicated, and geocoded to state centroids with random jitter for visualization. Agency data from CMS was geocoded from zip codes using the zipcodes package (bundled USPS data). Ownership was classified into For-Profit, Non-Profit, Government, Combination, and Other categories from the CMS Ownership field.

Maps built with deck.gl (GPU-accelerated WebGL layers) on CARTO Dark Matter basemap tiles. Data processing in Python (pandas, zipcodes). Static charts in matplotlib.
